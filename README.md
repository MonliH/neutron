# Neutron
Neutron is a programming language insspired by C, Python, and Java.

# Features
- [x] Classes
- [x] Functions
- [x] Built-Ins (e.g. print, get, stdin)
- [x] Primitive Types (Integers, Floats, Booleans, Strings)
- [x] Python Lists, Tuples
- [x] **Numpy  arrays Builtin!!**
- [ ] For Loops
- [x] Whille loops
